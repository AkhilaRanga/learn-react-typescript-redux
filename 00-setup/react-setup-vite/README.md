#### Typescript setup (with best practices)

1. Create a new react project with vite

```
npm create vite@latest <project-name>
# Prompt questions
# Select a framework: React
# Select a variant: Typescript
# Install with npm and start now?
## If selected Yes, open http://localhost:5173/ in browser
## If selected No, run below command and then open http://localhost:5173/ in browser
npm install && npm run dev
```

2. README generated by vite suggests making some changes to eslint config

```js
export default defineConfig([
  globalIgnores(["dist"]),
  {
    // ...
    extends: [
      // ...
      // Replace tseslint.configs.recommended with this
      tseslint.configs.recommendedTypeChecked,
    ],
    languageOptions: {
      parserOptions: {
        project: ["./tsconfig.node.json", "./tsconfig.app.json"],
        tsconfigRootDir: import.meta.dirname,
      },
      // ...
    },
  },
]);
```

3. Install prettier and update eslint config

```
npm install -D prettier eslint-config-prettier
```

```js
import prettier from "eslint-config-prettier";
export default defineConfig([
  globalIgnores(["dist"]),
  {
    // ...
    extends: [
      // other configs...
      prettier,
    ],
  },
]);
```

4. Add scripts to package.json

```
  "scripts": {
    ...
    "lint:fix": "eslint . --fix",
    "format": "prettier --write ."
  },
```

```
npm run lint:fix
npm run format
```

5. Setup pre commit git hook with husky

```
npm install husky --save-dev
npx husky init # adds script prepare in package.json and creates .husky folder
npm run prepare
# Update .husky/pre-commit file to have 'npm run lint:fix'
chmod +x .husky/pre-commit
git commit -m "Test"
```
